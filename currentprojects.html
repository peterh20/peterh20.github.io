Rocket Project







I designed a model rocket with a telemetry system which guides itself towards a inputted GPS target.

It took alot of time to complete this project, as the entire project was made from scratch:
- 3d printed parts were made using Solidworks.
- Custom circuit board was made using kiCAD.
- Canard fins were controlled using data from the GPS and accelerometer.
- The teensy microconroller with the flight program was coded in C++
- Data recorded on the sd card was analyzed in Matlab.
